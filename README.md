# bookeditor-starter

A minimal Node.js + TypeScript starter project with Express, ESLint, Prettier, Jest and Docker.

Quick start

Install dependencies:

```bash
npm ci
```

Run in dev mode:

```bash
npm run dev
```

Build and start:

```bash
npm run build
npm start
```

Run tests:

```bash
npm test
```

Build Docker image:

```bash
npm run docker:build
```
